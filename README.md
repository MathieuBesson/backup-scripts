# Backup des serveurs

## Pré-requis

Définir les varaibles d'env sur le serveur

Il faut que le lanceur est un accès ssh avec clé sur le serveur avec l'user root

Ajouter le fichier var.sh sur le serveur en se basant sur var.dev.sh
